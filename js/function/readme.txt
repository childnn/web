函数声明与函数表达式:
  函数声明:
          function name() {
             // ....
          }
  函数表达式:
         var name = function() {
            // ...
         }
  1. 在处理其他代码之前, 先处理 函数声明.
  2. 函数表达式随其他代码一起被处理.
  3. 函数声明 不返回指向函数的引用; 而是创建(浏览器引擎创建)一个与函数同名的变量, 并将指向函数的引用赋给它.
  4. 函数表达式 返回一个引用,该引用指向函数表达式创建的函数.
  5. 可在变量中存储指向函数的引用.
  6. 函数声明是完整的语句, 而函数表达式只是语句的一部分.
  7. 对于函数生命和函数表达式创建的函数,用相同的方式处理调用它们的语句.

  一等值:
    1. 将其赋值给变量或存储在数组和对象等数据结构中.
    2. 将其传递给函数.
    3. 从函数中返回它们.
