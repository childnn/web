event: 对应的方法就是 on+时间名
    resize: 当用户调整浏览器窗口大小时触发.
    play: 用户单击网页 <video> 标签播放按钮时触发
    pause: 用户单击 <video> 元素暂停按钮时触发
    unload: 用户关闭浏览器窗口或切换到其他网页时触发
    load: 浏览器加载网页完毕后触发
    dragstart: 用户拖拽网页元素时触发
    drop: 用户放下拖拽的元素时触发
    mousemove: 在元素上移动鼠标时触发
    mouseover: 用户将鼠标指向元素时触发
    keypress: 用户按下任意键触发
    mouseout: 用户将鼠标从元素上移开时触发
    touchstart: 在触摸设备上,用户触摸并按住元素时触发
    touchend: 用户停止触摸时触发
